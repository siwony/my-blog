---
```bash
brew install mysql
```
를하면 자동으로 설치가 될것이다.  
설치를 확인하기 위해 버전을 확인해보자
```bash
mysql --version
```
2020, 12, 17 일 기준으로 8.0.22 버전이 설치될것이다.
