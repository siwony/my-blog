### 예시

```bash
lsof -i TCP:PORT // ex) lsof -i TCP:8080
kill -9 PID    // ex) kill -9 18234
```
> PORT : PORT 번호  
> PID : `lsof -i TCP:PORT`를 통해 구한 PID
1. PORT번호로 실행중인 프로세스의 PID를 구한다.
2. 위에서 PID를 통해 프로세스를 강제종료한다.
