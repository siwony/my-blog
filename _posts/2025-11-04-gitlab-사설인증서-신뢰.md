GitLab은 ssl인증을 하기 위해 사설인증서를 사용합니다. 따라서 ssl 관련 통신을 하는경우 경고 혹은 통신이 안되는 경우를 해결하기 위해 해당 문서를 작성합니다.

# 사설인증서 신뢰 - Windows

1. 제어판 → 컴퓨터 인증서 관리
2. 신뢰할 수 있는 루트 인증기관 → 인증서 우클릭 → 모든 작업 → 가져오기 클릭
    
    ![인증서 가져오기 메뉴](/assets/img/posts/2025-11-04-gitlab-사설인증서-신뢰/CleanShot_2025-11-04_at_09.41.412x.png)
    
3. 인증서 가져오기 마법사에서 인증서 추가하기
    1. 다음버튼 클릭
        
        ![마법사 시작](/assets/img/posts/2025-11-04-gitlab-사설인증서-신뢰/CleanShot_2025-11-04_at_09.43.182x.png)
        
    2. "찾아보기"로 원하는 인증서 선택 후 다음
        
        ![인증서 선택](/assets/img/posts/2025-11-04-gitlab-사설인증서-신뢰/CleanShot_2025-11-04_at_09.43.402x.png)
        
    3. 다음버튼 클릭
        
        ![저장소 선택](/assets/img/posts/2025-11-04-gitlab-사설인증서-신뢰/CleanShot_2025-11-04_at_09.43.562x.png)
        
    4. "마침" 클릭
        
        ![마법사 완료](/assets/img/posts/2025-11-04-gitlab-사설인증서-신뢰/CleanShot_2025-11-04_at_09.44.012x.png)
        
4. GitLab 접속 후 경고 사라졌는지 확인 (안사라지면 브라우저 재부팅 혹은 PC 재부팅)

![GitLab 접속 확인](/assets/img/posts/2025-11-04-gitlab-사설인증서-신뢰/CleanShot_2025-11-04_at_09.44.422x.png)

# MacOS

1. keychain access(키체인 접근) 실행
2. 시스템 → 인증서에 신뢰하고자 하는 인증서 끌어넣기
    
    ![키체인 인증서 추가](/assets/img/posts/2025-11-04-gitlab-사설인증서-신뢰/CleanShot_2025-11-04_at_09.50.19.png)
    
3. 추가한 인증서 더블클릭 후 "이 인증서 사용 시" 에서 항상 신뢰로 변경
    
    ![인증서 신뢰 설정 1](/assets/img/posts/2025-11-04-gitlab-사설인증서-신뢰/CleanShot_2025-11-04_at_09.52.53.png)
    
    ![인증서 신뢰 설정 2](/assets/img/posts/2025-11-04-gitlab-사설인증서-신뢰/CleanShot_2025-11-04_at_09.53.33.png)
