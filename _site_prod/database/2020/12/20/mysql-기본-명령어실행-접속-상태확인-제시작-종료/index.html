MySQL 기본 명령어(실행, 접속, 상태확인, 제시작, 종료)

  • 실행

    ```bash $ mysql.server start

  • 접속

      $ mysql -u root -p

    비밀번호가 없으면 -p는 생략이 가능하다.

  • 상태확인

      $ mysql.server status

  • 재시작

      $ mysql.server restart

  • 종료

      $ mysql.server stop
