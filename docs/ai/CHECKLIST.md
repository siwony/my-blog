# ✅ AI Quick Checklist

작업 시작 전·중·후에 공통으로 확인해야 할 사항을 1페이지로 모았습니다. 세부 기준은 [`AI_DEVELOPMENT_GUIDELINES.md`](../guidelines/AI_DEVELOPMENT_GUIDELINES.md)를 따릅니다.

## 🏁 시작 전
- [ ] 작업과 연관된 가이드라인/스펙 확인 (`docs/` → 필요 문서 링크는 `DOC_MAP.yml` 참조)
- [ ] 변경 범위와 테스트 전략 선정 (단위/통합/스냅샷 여부)
- [ ] 커밋 단위 설계: 하나의 목적·원자적 변경·명확한 메시지

## 🔧 진행 중
- [ ] 코딩·문서 작성 시 관련 규칙 준수 (스타일, 메타데이터, 접근성)
- [ ] 커밋을 세분화하고 WIP 코드 미포함
- [ ] 테스트 작성/수정 및 실행 (플레이북에 명시된 최소 세트)

## 🏁 마무리
- [ ] 문서 업데이트 (아키텍처/가이드/README) 여부 점검
- [ ] 로컬 빌드/테스트 통과 확인 로그 남김
- [ ] 다음 작업자를 위한 노트/주석 최소화된 상태로 정리
