# 🤖 AI Ops Hub

레포를 읽고 작업하는 AI(도우미·에이전트)가 **항상 여기서 시작**하도록 하는 단일 진입점입니다. 사람이 읽기에도 간결하지만, 머신이 참조하기 쉽게 링크와 맥락을 압축해 두었습니다.

## 🚀 빠른 시작
1. **체크리스트**: [`CHECKLIST.md`](./CHECKLIST.md)에서 전/중/후 단계 확인
2. **작업별 플레이북**: [`TASK_PLAYBOOK.md`](./TASK_PLAYBOOK.md)에서 맡은 일에 맞는 절차 선택
3. **원본 레퍼런스**: 필요한 경우 아래 "핵심 참조" 문서를 연계 확인

## 🗂️ 구성
- [`CHECKLIST.md`](./CHECKLIST.md): 1페이지 짜리 공통 점검표
- [`TASK_PLAYBOOK.md`](./TASK_PLAYBOOK.md): 기능 추가, 버그 수정, 문서 갱신, 스타일 작업용 단계별 가이드
- [`DOC_MAP.yml`](./DOC_MAP.yml): `docs/` 전역 문서를 태그/요약과 함께 나열한 머신 친화적 인덱스

## 🧭 사용 흐름 (요약)
1) **CHECKLIST**로 기본 준수 사항 확인 → 2) **TASK_PLAYBOOK**에서 해당 시나리오 절차 수행 → 3) 필요 시 **핵심 참조** 문서로 세부 확인 → 4) 변경 후 테스트/문서 업데이트.
